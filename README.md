# Angular + Spring Boot Employee Management System 👨‍💼📊

This project is a **simple Employee Management REST API** built with **Spring Boot**, designed to be consumed by an **Angular frontend**.  
It supports full **CRUD operations**, **pagination**, and **search functionality**, making it a solid beginner-to-intermediate full-stack project.

---

## ✨ Features

- Create new employees
- View employees with **pagination**
- Search employees by **name or email**
- Update employee details
- Delete employees
- RESTful API design
- CORS configuration for Angular (`localhost:4200`)
- Clean layered architecture (Controller, Service, Repository, Entity)

---

## 🛠️ Technologies Used

- **Backend:** Java, Spring Boot
- **Database:** Spring Data JPA / Hibernate  
- **Security:** Spring Security (CSRF disabled for API usage)
- **Frontend (separate project):** Angular
- **Build Tool:** Maven
